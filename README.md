# TP Servlet – Apache Tomcat

## Objectif

Créer une application web Java utilisant un servlet déployé sur Apache Tomcat.

## Technologies

- Java 17+
- Maven
- Apache Tomcat 10.1
- IntelliJ IDEA Community

## Exécution

1. mvn clean package
2. Copier le fichier WAR dans le dossier webapps de Tomcat
3. Lancer Tomcat (startup.bat)
4. Accéder à :
   - http://localhost:8080/tp-servlet-tomcat-1.0-SNAPSHOT/
   - http://localhost:8080/tp-servlet-tomcat-1.0-SNAPSHOT/hello
